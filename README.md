# ATmega4809-A-breakout

48-pin DIP breakout board for the Microchip ATmega4809-A microcontroller.

![Kicad render - front](Images/ATmega4809-A-breakout-render-front.png "Kicad render - front") Kicad render - front.

![Kicad render - back](Images/ATmega4809-A-breakout-render-back.png "Kicad render - back")Kicad render - back.

AVCC is connected to VCC (via an inductor). Sorry about that, analogue fans, but I ran out of pins.
